## ğŸ” Autentikasi

Untuk bind ke server LDAP:

```bash
ldapsearch -x -H ldap://localhost \
  -D "cn=admin,dc=example,dc=com" \
  -w admin \
  -b "dc=example,dc=com"
```

Penjelasan:

| Opsi | Fungsi                        |
| ---- | ----------------------------- |
| `-D` | Bind DN (login sebagai siapa) |
| `-w` | Password                      |
| `-b` | Base DN pencarian             |
| `-H` | Host atau alamat LDAP server  |
