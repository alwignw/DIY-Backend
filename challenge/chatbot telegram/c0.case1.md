## ğŸ§ª Contoh: Cek IP di Spamhaus dengan DNS

```python
import dns.resolver

def check_spamhaus(ip):
    reversed_ip = ".".join(ip.split(".")[::-1])
    query = f"{reversed_ip}.sbl.spamhaus.org"
    try:
        result = dns.resolver.resolve(query, 'A')
        return True
    except dns.resolver.NXDOMAIN:
        return False
```
